I built this Defile Calculator together with my colleague Jianing Cai, in which I designed and implemented the Dynamic Programming recursion Algorithm and logic of minion interactions. This Calculator takes the current board status and outputs the sequences of steps to reach a Defile-Clearable board. User can also click to see the board status lively after each step.

I am a 5-year Hearthstone player, and I hope this Calculator will help Hearthstone players to make wise decisions when they have Defile card in hand! In the future, I plan to build a Machine Learning model so that the user wouldn't need to type in all the numbers but only upload a screenshot.
